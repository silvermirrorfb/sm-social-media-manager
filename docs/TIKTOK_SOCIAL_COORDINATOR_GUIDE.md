# TikTok Dashboard Guide For Social Media Coordinator

This guide explains how to use the Silver Mirror TikTok dashboard for daily work.

## What the dashboard is for

The dashboard helps you manage three kinds of TikTok work:

1. Replying to inbound DMs.
2. Sending outreach DMs to influencers.
3. Reviewing negative comments and tracking what action was taken.

The dashboard is a planning and tracking tool. It does not complete the final TikTok action for you yet. You will still do the final reply, outreach message, or comment action manually inside TikTok.

The dashboard can now generate a draft suggestion for a saved task. You still review the draft and complete the final action manually in TikTok.

## Before you start

At the start of your shift:

1. Open the Silver Mirror dashboard.
2. Open TikTok in the same browser.
3. Log into TikTok manually.
4. Go back to the dashboard.
5. Click the button confirming that you are logged into TikTok.

If the dashboard shows a TikTok login reminder, do not work the queue until you have logged into TikTok.

## What you will see on the dashboard

The dashboard has:

- a TikTok login reminder
- queue status and environment status cards
- an "Add task" form
- three work lanes:
  - Inbound DMs
  - Influencer Outreach
  - Negative Comment Review

## How to add a new task

Use the "Add task" form whenever you want to track work that needs to happen in TikTok.

Fill in:

- `Workflow`
  - Choose `Inbound DMs`, `Influencer Outreach`, or `Negative Comment Review`.
- `Priority`
  - Use `High` for urgent or sensitive items.
- `Handle`
  - Enter the TikTok username if known.
- `Author / owner`
  - Enter the person who sent the DM or posted the comment, if useful.
- `Message or comment`
  - Paste the full DM, outreach brief, or comment text.
- `Suggested reply`
  - Paste or write the draft reply you want to use.
- `Suggested action`
  - Describe what should happen in TikTok.
- `Action URL`
  - Paste the TikTok URL if you have one.
- `Assigned to`
  - Put your name or the name of the person handling it.
- `Operator note`
  - Add context, concerns, or escalation notes.

Then click `Add to queue`.

## How to work Inbound DMs

Use this lane for messages that came into TikTok from customers or prospects.

Typical workflow:

1. Open the task.
2. Review the DM text.
3. Draft or update the suggested reply.
4. If needed, click `Generate Draft` to create a first pass.
5. If the draft looks good, click `Copy Draft` and paste it into TikTok.
6. Open TikTok and find the conversation.
7. Send the reply manually.
8. Update the task status.
9. Write the outcome.

Suggested statuses:

- `new` when it has not been touched yet
- `drafting` when you are preparing the response
- `ready` when the draft is approved and ready to send
- `done` after the message has been sent
- `blocked` if you need help, approval, or more information

## How to work Influencer Outreach

Use this lane for outbound creator and influencer messaging.

Typical workflow:

1. Add the creator handle.
2. Add the outreach message draft.
3. Add any note about why the creator is a fit.
4. Click `Generate Draft` if you want a first-pass outreach message.
5. Click `Copy Draft` if you want to paste the message into TikTok.
6. Open the creator profile in TikTok.
7. Send the outreach manually.
8. Update the outcome.

Examples of outcomes:

- `sent`
- `waiting on reply`
- `no DM option available`
- `escalated to team`

## How to work Negative Comment Review

Use this lane for comments that may need to be hidden, removed, reported, or escalated.

Typical workflow:

1. Paste the comment text into the queue.
2. Add the TikTok URL if available.
3. Write the suggested action.
4. Click `Generate Draft` if you want help deciding the safest action.
5. Open TikTok and review the full context.
6. Take the action manually in TikTok.
7. Record exactly what happened in the `Outcome` field.

Examples of outcomes:

- `removed`
- `hidden`
- `left up after review`
- `escalated to manager`
- `needs legal review`

## When to use Blocked status

Mark a task as `blocked` if:

- you are not sure what reply to send
- the message is sensitive or high-risk
- the comment may need management review
- you cannot access the right TikTok page
- there is a login/session issue

Always leave a clear note when you use `blocked`.

## What to write in Operator Note

Use `Operator note` for short, practical context such as:

- why you changed the draft
- what risk you noticed
- who approved the message
- what still needs follow-up
- why the task was blocked

Keep notes clear and short so the next person can continue quickly.

## End of shift checklist

Before ending your shift:

1. Make sure every task you touched has the correct status.
2. Fill in the `Outcome` field for completed work.
3. Leave an `Operator note` on anything unfinished or blocked.
4. Confirm high-priority items are either done or clearly assigned.

## Known limitations right now

The dashboard currently does not:

- log into TikTok for you
- automatically read TikTok messages or comments
- automatically send replies or outreach
- automatically remove comments

You should think of it as the command center and record of work, while TikTok is still where the action happens.

## If something looks wrong

If the queue is empty when it should not be, or the dashboard shows config issues:

1. Do not guess.
2. Leave a note for the team.
3. Flag the issue as blocked.

If Google Sheets is disconnected, the dashboard may not save tasks correctly. In that case, pause and escalate rather than working without a record.
